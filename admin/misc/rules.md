# Team Charter (NN Group Template)

## Personal Goals

- Get a job
- Build genuine bonds with teammates
- Strengthen web development skills

## Values, Mission, and Common Goals

- Gain experience working in a team environment
- Learn how to communicate effectively in a large group
- Develop a project to showcase
- Maintain a positive mindset while working together
- Stay open-minded
- Be coachable and open to failure
- Be vulnerable with each other and build mutual trust

## Strengths

- Some members have web dev / internship experience
- Good communication skills

## Weaknesses

- Some members aren't familiar with CSS
- Vibecoding as a strength, but a weakness if relied on too heavily
- Procrastination

## People, Roles, and Accountabilities

- Open to changing roles, from pairs to more specialized coding
- Team leads focus more on organizing teams
- Start with mobbing, move to pairing, then soloing

## Needs and Expectations

- Members expected to regularly communicate availability on Slack
- Expect teammates to offer help respectfully
- Willingness to help each other
- Communicate if falling behind and ask for help
- Report repo issues immediately

## Rules and Activities

- Check Slack multiple times per day
- Finish tasks well before the due date to account for potential problems
- Attend weekly meetings (discussion and possibly one online)
- Use AI wisely

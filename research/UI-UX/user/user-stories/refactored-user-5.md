# User Story 3: The "Director's Dashboard" Workflow

## Story
As a student, I want a collaborative interface where the AI acts as a "Junior Assistant" that constantly checks in for guidance, so that the workflow feels like a partnership rather than an autonomous process.

## Why This Matters
The tracker should feel like a chat between a director and a worker. This prevents the "slopfest" by ensuring that the student is constantly "pinged" for small bits of direct input, keeping the project on a tight, human-controlled path.

## Acceptance Criteria
- The UI includes a "Quick Feedback" bar (e.g., "Good," "Change Direction," "Needs more detail").
- All AI actions are logged in a way that shows which specific human instruction prompted them.
- The system prioritizes human-written notes over the AI's internal "learned" behavior.

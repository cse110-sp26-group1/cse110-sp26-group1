Project Description:
AIT will be an issue tracker built from the ground up with AI in mind. It will be designed to work for both humans and AI agents, keeping things fast, simple, and easy to integrate into an AI workflow. At its core it will be a CRUD application for creating, reading, updating, and deleting issues, integrated with AI features.
